#+TITLE: Anthropic instala secretamente software espía al instalar Claude Desktop
#+OPTIONS: toc:nil num:nil

Buenos días.

El artículo [[https://www.thatprivacyguy.com/blog/anthropic-spyware/][«Anthropic instala secretamente software espía al instalar Claude Desktop»]] relata el hallazgo de un archivo no autorizado en un MacBook, generado por la instalación de Claude Desktop. El usuario, al revisar los archivos del navegador Brave, identificó un manifiesto de mensajería nativa que habilita la ejecución de código local sin su permiso.

El archivo, ubicado en una ruta oculta, está configurado para permitir que extensiones de navegador ejecuten un binario de Claude con privilegios de usuario. Aunque el usuario nunca instaló extensiones de Anthropic, el archivo fue creado tras instalar la aplicación de escritorio, sin notificación previa.

El caso expone un problema de transparencia y seguridad, ya que el software opera sin consentimiento explícito. El usuario denuncia la falta de aviso y el potencial riesgo de que aplicaciones instalen componentes ocultos que afecten la privacidad.

Saludos, José A.
